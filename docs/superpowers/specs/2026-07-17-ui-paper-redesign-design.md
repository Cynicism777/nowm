# nowm 前端 UI 美化（纸感简约）— 设计文档

- 日期：2026-07-17
- 项目：nowm
- 关联试验场：`experiment/`（gitignored，仅本地预览）
- 参考预览：`experiment/index.html`（首页 / 工具页 / Locked）

## 1. 背景与目标

现有前端已具备可用结构（毛玻璃顶栏、圆角按钮、轻量弹簧动效），但整体偏「灰白原型」：
品牌文案「cynic 工具箱」在页面上不自然，背景无层次，Locked 页过空。

**目标：** 在不增加页面复杂度的前提下，升级为「柔和纸感 + 微纹理」的简约成品感；品牌统一为 **`nowm`**。

**非目标：**
- 不新增功能、不改后端 API
- 不加插画墙、统计条、多入口堆砌、复杂导航
- 不做深色主题、不做分享 OG 图（本期）

## 2. 已确认决策

| 项 | 选择 |
|----|------|
| 气质 | B · 柔和纸感 + 微纹理纸纹 |
| 品牌 | 纯词标 `nowm`（无「cynic」、无中文主标题） |
| 首页副文案 | 可保留一行短句（试验场：`无水印，干净取图`）；可随时删减 |
| 静态资源策略 | 沿用 SVG 纸纹；建议补 favicon；不需要插画/照片 |
| 视觉验收 | 以 `experiment/` 静态稿为基准，正式实现应对齐观感 |

## 3. 视觉系统

### 3.1 颜色（token）

| Token | 约值 | 用途 |
|-------|------|------|
| `--bg` | `#f1ece4` | 页面底色 |
| `--bg-deep` | `#e8e1d6` | 底色渐变末端 |
| `--ink` | `#2a2824` | 主文字 |
| `--muted` | `#7a746a` | 次要文字 |
| `--surface` | `rgba(255,252,247,0.78)` | 顶栏毛玻璃 |
| `--surface-solid` | `#fffdf9` | 卡片 / 输入框 |
| `--line` | `rgba(42,40,36,0.08)` | 细分割线 |
| `--accent` | `#2f4a6e` | 主按钮 / 链接（墨蓝，避免高饱和蓝与紫色） |
| `--danger` | `#b42318` | 错误文案 |

### 3.2 字体

- **词标：** Syne（800），大字号、字距略收（`letter-spacing: -0.06em`）
- **正文 UI：** Noto Sans SC（外网可用时）或系统中文字体回退
- **部署约束：** 若生产环境不便拉 Google Fonts，词标可改为本地 woff2；正文退回 `-apple-system` / `PingFang SC` / `Microsoft YaHei`

### 3.3 纹理与材质

- 全页背景：浅暖渐变 + 可平铺纸纹
- 纸纹默认：`paper-texture.svg`（fractalNoise，低对比、`multiply` 叠加）
- 可选替换：用户提供的浅棕灰、低对比、可平铺 PNG（约 512×512）
- 卡片：近白实底 + 轻阴影；输入框同系表面色

### 3.4 动效

- 保留：按钮 / 卡片按压缩放、工具卡 hover 微抬、图格入场
- 尊重 `prefers-reduced-motion` / `prefers-reduced-transparency`
- 不增加页面级花哨转场

## 4. 页面结构

### 4.1 首页

一屏一件事：

1. 顶栏：小字 `nowm`（半透明），无其他导航
2. 中央超大词标 `nowm`
3. 一行短副文案（可选；默认「cynic·百宝箱」）
4. **一个**工具入口卡片：「小红书无水印下载」+ 短描述

禁止：多卡片墙、徽章、统计、插画。

### 4.2 工具页（`/xhs`）

- 顶栏：`‹ 返回` + 短标题「小红书无水印」
- 解析行：输入框 +「解析」
- 结果：标题截断、`@作者 · N 张`、主按钮「保存全部」/「打包下载 ZIP」（文案仍由 `save.ts` 能力探测决定）
- 图格：3:4；角上半透明「保存」胶囊
- 错误：一行红色文字，无模态

功能行为与现有一致，只改视觉与文案长度。

### 4.3 Locked（未授权）

- 顶栏小字 `nowm`
- 中央中等词标 + 两行说明（邀请链接）
- 无插画、无表单、无二维码

### 4.4 加载态

当前 `App` 在 auth 未就绪时 `return null`。本期可做**极轻**改进：同纸面背景上的空壳或微弱词标，避免白闪；不做骨架屏堆砌。

## 6. 静态资源清单

| 资源 | 必须 | 说明 |
|------|------|------|
| `paper-texture.svg` | 是（有现成） | 从 `experiment/` 迁入 `frontend/public/` 或 `frontend/src/assets/` |
| Favicon（SVG 或 32×32 PNG） | 建议 | 单色 `n` / `nowm` 缩写；深墨或透明底 |
| 真实纸纹 PNG | 否 | 仅当 SVG 质感不满意时替换 |
| 插画 / 摄影 | 否 | 本期不做 |
| OG 分享图 | 否 | 本期不做 |

## 7. 实现范围（前端）

触及文件（预期）：

- `frontend/index.html` — title / favicon
- `frontend/src/theme.css` — token、纸纹背景、通用控件样式
- `frontend/src/pages/Home.tsx` / `Locked.tsx` / `XhsTool.tsx` — 结构与文案
- `frontend/src/components/ToolCard.tsx` / `ImageGrid.tsx` — 样式对齐纸感
- 新增资源：纸纹、favicon
- 字体引入方式（link 或自托管）在实现计划中二选一

**不改：** `api.ts`、`auth.ts`、`save.ts` 业务逻辑；后端与 Docker。

## 8. 验收标准

1. 打开首页，第一眼是大词标 `nowm`，无「cynic 工具箱」字样
2. 背景可见低对比纸纹，整体暖浅而非冷灰
3. 工具页 / Locked 与试验场观感一致（允许字体加载差异）
4. 现有解析、单张保存、批量保存/ZIP、邀请鉴权行为不变
5. `prefers-reduced-motion: reduce` 下无持续动画干扰

## 9. 风险与回退

- **外网字体失败：** 回退系统字体，词标略减辨识度但可用
- **纸纹性能：** SVG 平铺 normally 极轻；若低端机卡顿，可降低 opacity 或去掉 `::after` 层
- **试验场与正式站偏差：** 以 experiment 为视觉基准做对照验收
